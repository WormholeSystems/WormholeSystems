# Getting help

Stuck, found a bug, or want to suggest something? Here's where to go.

## Discord

Our **Discord** is the fastest way to get help — the community and the developers are both there, and it's where new features and changes are discussed. You'll find a **Join Discord** button at the bottom of every page in this documentation.

Good for: setup questions, [self-hosting](/documentation/contributing-and-self-hosting/self-hosting) help, feedback, and just talking shop.

## GitHub issues

For reproducible **bugs** and concrete **feature requests**, open an issue on the [main repository](https://github.com/WormholeSystems/WormholeSystems). A clear issue is easier to track and fix than a message that scrolls away in chat.

## This documentation

A lot of "how does X work" questions are answered right here. Use the sidebar to browse, and remember most things on the map are [right-clickable](/documentation/getting-started/right-click-menus) — the menu is usually where the answer is.
