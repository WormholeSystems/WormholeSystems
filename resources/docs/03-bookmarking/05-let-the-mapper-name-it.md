# Let the mapper name it

You almost never need to type a bookmark name by hand. The mapper builds names in the [agreed format](/documentation/bookmarking/why-this-system) straight from the chain, so they're correct and consistent every time.

## Copy name

Right-click the connection between two systems and open **Copy name**. You'll see the ready-made bookmark for **each end** of the connection — pick the side you're sitting in and it's copied to your clipboard. Open the in-game bookmark dialog and paste.

This is the safest way to name a bookmark: no typing, no typos, and it always matches the map.

## Copy bookmark on jump

If you'd rather have it happen automatically as you fly, turn on **Copy Bookmark** in **Map settings → Mapping**. Then, whenever you record a jump by selecting the signature you came through, the bookmark for that hole is copied to your clipboard for you (with a small confirmation).

> _"Copy the connection bookmark name to your clipboard after selecting the signature you jumped through."_

## Prompt for signature

For the smoothest workflow, also enable **Prompt for Signature** in **Map settings → Mapping**. When it's on, the mapper asks which signature you jumped through each time you change system — turning logging the chain into a single click. Make sure your **active character** is the one doing the scanning so the prompt follows you.

With both settings on, mapping and bookmarking become almost a single motion: jump, pick the signature, paste the bookmark.
