# System status

Mark each system's **status** to colour the chain by intel, so the whole group can read the situation at a glance.

| Status        | Typical use                                               |
| ------------- | --------------------------------------------------------- |
| **Active**    | A system you're working — scanning, rolling, or watching. |
| **Friendly**  | Blues, allies, or your own staging.                       |
| **Hostile**   | Known reds or a system to avoid.                          |
| **Empty**     | Nobody home; cleared.                                     |
| **Unscanned** | Not yet probed.                                           |
| **Unknown**   | Status not assessed.                                      |

Status is a shared, visual signal. A glance at the map tells you where the danger is, where home is, and what still needs scanning — without reading a single note.
