# Adding systems & connections

Most of the time the chain builds itself as you fly (see [Mapping as you fly](/documentation/getting-started/auto-mapping)), but you can always add things by hand. Both actions need **Member** access or higher.

## Add a system

**Right-click an empty part of the map** and choose **Add Solarsystem**. A search dialog opens:

1. Start typing a system name in the search box.
2. Matches appear under **Search Results**; systems already on the map are listed separately under **Already in Map** so you don't add duplicates.
3. Click a result and it's dropped onto the map where you right-clicked.

## Add a connection

**Drag from one system to another.** Press and hold on the first system, drag to the second, and release — the connection is drawn between them. The app even sets a sensible **maximum ship size** automatically based on the systems involved (for example frigate-only for a C13, medium for a Turnur or Thera high-sec link), which you can adjust afterward from the connection's [right-click menu](/documentation/getting-started/right-click-menus).

## Tidy up

The empty-background right-click menu also helps you keep things neat:

- **Delete selection** — remove the systems you've selected.
- **Organize selection** — auto-arrange selected systems, with an adjustable spacing.
- **Clear map** — remove every unpinned system at once (pinned systems stay; see [Pinning](/documentation/organizing-your-map/pinning-and-ignore-list)).

These, and far more, live in the map's context menus — which is the next thing worth knowing about.
